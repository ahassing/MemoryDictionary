# MemoryDictionary
Coding Challenge for Spreetail technical interview. 

# Build Information
Requirements:  
* Apache Maven 3.6.3  
* Java version: >= 9  
In the root directory, run `mvn clean package`.

# Running the Application
`java -cp target/memorydictionary-1.0-SNAPSHOT.jar com.hassing.dictionary.App`
